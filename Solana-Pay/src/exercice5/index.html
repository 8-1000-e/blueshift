Solana Pay - Transaction Request

Scanne ce QR code avec Solflare (en mode Devnet)

Comment ça marche :

  1. Le wallet scanne le QR
  2. Il fait un GET vers le serveur → affiche "Blueshift demo"
  3. Tu confirmes → POST avec ton adresse
  4. Le serveur construit la transaction
  5. Tu signes → 1 SOL envoyé
